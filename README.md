# Начало начал...

### *Проект представляет собой программу, выполняющую учет ПО пользователей. Это необходимо для пользователей.*
#### *На данный момент в разработке, README будет дополняться.*

Предпологается, что программа будет хранить в себе список установленных и неустановленных приложений на устройстве пользователя, а также получать сведения о новых обновлениях. Пользователи, на свое усмотрение, смогут установить, обновить или удалить ПО.
